# 新闻覆盖分析 & 增强方案

## 用户反馈的两个新闻案例分析

### 案例1: 伊朗协议 + 霍尔木兹海峡船只回升

**新闻内容**: 伊朗与部分国家达成协议，霍尔木兹海峡通行船只数量回升

**原有系统覆盖情况**: ⚠️ 部分覆盖
- ✅ 已覆盖: 伊朗、制裁、地缘冲突
- ❌ 缺失: 霍尔木兹海峡名称、船只通行量、供应通道恢复

**新闻分类**: 地缘政治 (Geopolitics) - TIER1高冲击

**影响因子分析**:
| 维度 | 评估 |
|------|------|
| 直接冲击 | 全球原油供应中断风险缓解 |
| 影响路径 | 霍尔木兹海峡→全球原油供应→油价→能源股情绪 |
| 中国煤炭相关性 | 0.6 (中等，通过油价传导) |
| 影响持续时间 | 3-7天 |
| 情绪 | 正面 (供应风险下降) |

**系统增强**:
```python
# 新增关键词
"Strait of Hormuz", "Hormuz", "霍尔木兹海峡",
"shipping lane", "maritime traffic", "vessel traffic"

# 新增事件类型
"strait_of_hormuz_closure": 9.5分  # 关闭
"strait_of_hormuz_reopen": 7.0分   # 恢复 (正面)
```

### 案例2: 国内航线燃油附加费上涨

**新闻内容**: 国内航线燃油附加费上涨

**原有系统覆盖情况**: ❌ 未覆盖
- 原有查询集中在国际能源、OPEC、EIA
- 缺失国内具体行业政策追踪

**新闻分类**: 国内能源政策 (Domestic Energy Policy) - TIER2中冲击

**影响因子分析**:
| 维度 | 评估 |
|------|------|
| 直接冲击 | 航空出行成本上升 |
| 影响路径 | 燃油附加费→机票价格→航空需求→航空煤油需求→炼厂开工→煤炭替代 |
| 中国煤炭相关性 | 0.3 (较低，间接影响) |
| 影响持续时间 | 1-2周 |
| 情绪 | 中性偏负面 (成本上升) |

**系统增强**:
```python
# 新增关键词
"燃油附加费", "航空煤油", "jet fuel surcharge",
"国内航线", "发改委油价", "NDRC fuel price"

# 新增事件类型
"fuel_surcharge_increase": 5.5分
    影响逻辑: 航空成本上升→抑制出行→煤油需求下降→炼厂开工调整
```
