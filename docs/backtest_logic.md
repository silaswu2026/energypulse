# EnergyPulse 回测验证逻辑

## 核心架构：跨市场异步决策

### 时间线 (北京时间)

```
T-1日 21:30        T-1日 次日 04:00      T日 08:30         T日 09:30
(美盘中段)          (美股收盘)            (报告生成)        (A股开盘)
   │                    │                    │                 │
   ▼                    ▼                    ▼                 ▼
美国宏观数据  →   数据采集(06:00)   →   AI分析(08:30)  →   投资决策
发布(T-1)                                        ↓                 ↓
                                           报告生成              A股交易
                                           (09:00前)            实际收益
```

### 数据流向

| 时间 | 动作 | 数据来源 |
|------|------|---------|
| 06:00 | 采集美国数据 | FRED, FMP, EIA (T-1收盘) |
| 07:30 | 采集供应链 | mx-search (BDI/库存/日耗) |
| 08:00 | 采集中国数据 | Tushare (T-1 A股收盘) |
| 08:30 | AI分析+报告 | 综合T-1国内外数据 |
| 09:00 | 报告发布 | 用于T日09:30 A股决策 |

## 回测验证流程

### 1. T日 08:30 信号生成

AI基于T-1数据生成预测：
```
输入(T-1):
  美国10Y: 4.20% → 4.10% (-10bp)
  DXY: 104.5 → 103.8 (-0.7%)
  WTI: $80 → $82 (+2.5%)
  XLE: +1.8%

AI逻辑:
  美债↓ + 美元↓ + 油价↑ = 全球流动性宽松 + 能源涨价
  → 预测: 看涨中国煤炭 (置信度75%)

输出:
  信号: BULLISH
  建议: 买入中国神华/陕西煤业
  仓位: 50%
```

### 2. T日 15:00 实际结果

```
实际走势(T日):
  中国神华: +2.1%
  陕西煤业: +1.8%
  煤炭ETF: +2.0%

验证: ✅ 预测正确
收益率: +2.0%
```

## 关键指标

| 指标 | 说明 | 目标值 |
|------|------|--------|
| 胜率 | 正确预测 / 总预测 | >60% |
| 盈亏比 | 平均盈利 / 平均亏损 | >1.5 |
| 夏普比率 | 超额收益 / 波动率 | >1.0 |
| 最大回撤 | 最大连续亏损 | <10% |

## 信号强度分级

| 强度 | 条件 | 仓位 | 示例 |
|------|------|------|------|
| 🔴极强 | 3+指标共振 | 80% | 美债↓+美元↓+油价↑ |
| 🟠强 | 2指标共振 | 50% | 美债↓+地缘冲突 |
| 🟡中 | 1指标+叙事 | 30% | 仅美债↓ |
| ⚪弱 | 单一指标 | 0% | 油价小幅波动 |
